Hevelop Lesti_FPC-AOE_AsyncCache for Magento 1.x
=====================

This module is a layer of compatibilization between Lesti_FPC and AOE_AsyncCache.

Requirements
------------
- PHP >= 7.0.0 (could work with PHP5 but it's not tested)

Compatibility
-------------
- Magento >= 1.7.1;
